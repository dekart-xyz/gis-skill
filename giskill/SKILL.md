# GIS Skill

## Trigger

Use this skill when the task requires geospatial analysis, map creation, or spatial SQL support.

## Core behavior

- Clarify datasource and coordinate system before running analysis.
- Prefer reproducible SQL-based transformations over manual UI steps.
- Keep map outputs interpretable: clear titles, legends, and units.
- Highlight assumptions (projection, distance units, geometry validity).

## Output expectations

- Query or processing steps used.
- Map-ready result summary.
- Short interpretation of findings and limitations.
